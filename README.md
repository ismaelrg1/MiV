# Modelización y Visualización
## Práctica 1 : Visualización de Datos

### Instalar librerias
Instalar **librerias** necesarias para la ejecución del proyecto. 
```
$ pip install -r requirements.txt
```

### Ejecución del código

```
$ python3 ./visualización.py
```
